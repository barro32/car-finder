# CarFinder App Features

This document tracks all features of the CarFinder app. Please update this file whenever a new feature is added.

## Current Features

### 1. Report Stolen Car
- Users can report their car as stolen, including the location.

### 2. Notifications
- Users in the vicinity are notified of stolen cars nearby.

### 3. Map View
- Main UI is a map with pins for each missing car.


### 4. Multi-Platform
- Web-first app, also deployable to Android and iOS.

### 5. API Placeholder
- Basic API route set up for car-related features (to be expanded).
