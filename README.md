Car finder is a web first app, which is also available on Android and iOS.

The app helps people find their missing / stolen cars.
A user can report their car stolen at a location.
Other users will be notified of stolen cars in their vicinity.

The main UI is a map view and pins for each car that is missing.

The base tech is a nextjs v15 app.
It should also be deployable as an android and ios app.
